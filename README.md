22. Encuentre las resistencias mínima y máxima dentro de los límites de tolerancia para cada resistor del problema 21.

(a) rojo, violeta, naranja, oro

Resistencia: 27000 Ω

Tolerancia:  ±5%

5% de 27000 = 1350

Resistencia máxima: 1800 + 1350 = 28350 Ω

Resistencia mínima: 1800 - 1350 = 25650 Ω


(b) café, gris, rojo, plata

Resistencia: 1800 Ω

Tolerancia:  ±5%

5% de 1800 = 90

Resistencia máxima: 1800 + 90 = 1890 Ω

Resistencia mínima: 1800 – 90 = 1710 Ω
